# "同事件提及"关系精准化分类 — 决策流程与变更总结

## 一、问题定义

原始数据中214条关系被统一标记为"同事件提及"，仅表示两家公司出现在同一篇新闻中，但无法区分它们之间的具体语义关系。用户需要对这些关系赋予更精准的名称。

## 二、分析流程

### 第1步：数据提取

从 [cross_doc_dataset.json](front/public/cross_doc_dataset.json)（3.1MB）中提取所有 `relation == "同事件提及"` 的记录，得到214条关系，每条包含：

- company1 / company2：两家公司全称
- evidence：证据句
- news：新闻标题、来源、时间、正文、URL

### 第2步：数据结构摸底

发现数据特征：

- 214条关系仅涉及 **64篇独立新闻**
- 存在大量 A↔B 双向镜像关系（同一对公司在两个方向各存一条）
- 单篇新闻可产生多条关系（最多一篇产生22条，即11对公司各双向一条）
- 原始数据中关系是嵌套字典结构：`relations[公司A][公司B] = {relation, evidence, news}`

### 第3步：新闻标题模式识别

对64篇新闻标题做聚合分析，发现高频模式：

| 模式 | 典型标题 | 关系数 |
|------|---------|--------|
| IPO周报/审核 | "注册制IPO周报：6家注册生效" | 大量 |
| IPO被否名单 | "25药企欲发力IPO未成功" | 多条 |
| 监管处罚 | "【突发】24家械企被点名!" | 4条 |
| 业绩对比 | "宁波建工收盘下跌1.45%..." | 多条 |
| 行业分析 | "民营眼科医院慈善资金循环疑云" | 6条 |
| 公告汇总 | "YY \| 每日债市公告严选" | 2条 |

### 第4步：关键词模式挖掘

对证据句和新闻正文做关键词匹配，识别出以下语义维度：

- **IPO相关**（108条匹配）：IPO、上市、发行、注册、过会、中止、被否、终止
- **监管处罚**（22条）：点名、飞检、处罚、违规、红黑榜
- **行业对比**（22条）：领头羊、可比公司、市场份额
- **高管关联**（8条）：就职于、曾任、兼任、独立董事
- **业绩财务**（12条）：业绩、盈利、市盈率

### 第5步：典型案例深度分析

逐篇阅读了以下典型新闻原文，验证分类假设：

| 案例 | 原文分析 | 实际关系 |
|------|---------|---------|
| 爱尔眼科 ↔ 华厦眼科 | 同为眼科行业领头羊被对比 | 行业对比分析 |
| 爱威科技 ↔ 北京凯因 | 同在"IPO被否"名单中 | IPO被否 |
| 爱威科技 ↔ 汉桑科技 | 汉桑科技董事曾在爱威就职 | 高管履历关联 |
| 爱博诺德 ↔ 大博医疗 | 同被国家飞检组点名 | 监管处罚名单 |
| 飞南资源新闻（22条） | 安徽多家IPO公司并列列举 | 列举式并列提及 |

## 三、分类体系设计

### 设计原则

1. **可解释性**：每个类型名称直接反映关系的业务含义
2. **互斥性**：一条关系只归入一个类型，按优先级从具体到模糊排序
3. **颗粒度适中**：不过粗（回到"同事件提及"），不过细（每种仅1-2条则合并）
4. **基于原文证据**：分类依据是新闻原文中的客观事实，非主观推测

### 分类规则与优先级

规则按优先级从高到低执行，首次命中即返回：

```
1.  高管履历关联     — 关键词：就职于/曾任/兼任 + 董事/高管/独立董事
2.  法律纠纷         — 关键词：仲裁/诉讼/索赔 + 原告/被告/申请人
3.  IPO中止          — 标题含IPO周报 + 内容含"中止"
4.  IPO暂缓          — 标题含上会/暂缓 + 内容含"暂缓"
5.  IPO同批过会      — 标题含上会/过会 + 内容含"过会/通过"
6.  IPO同批上会      — 标题含上会 + 内容含"上会"
7.  IPO被否          — 内容含"IPO被否/未获通过"
8.  IPO终止          — 标题含"终止IPO" 或 内容含"终止审查/撤单"
9.  监管处罚名单     — 内容含"被点名/飞检/通报批评/行政处罚"
10. 诚信评级关联     — 内容含"红黑榜/诚信/黑名单"
11. 行业排名/榜单   — 标题含"百强/排名/表彰"
12. 中介机构关联    — 标题含"中介机构/保荐/会所/律所"
13. 券商保荐关联    — 内容含证券公司 + 核查意见/财务顾问
14. 业绩对比        — 标题含"业绩/盈利/市盈率"
15. 公告汇总        — 标题含"公告严选/日报/快报"
16. 行业对比分析    — 内容含"领头羊/可比公司/市场份额"
17. 审计机构关联    — 标题含"续聘" + 内容含"审计机构"
18. 环保履约关联    — 内容含"碳市场/排污/环保"
19. 招标竞标关联    — 内容含"招标/中标/资格预审"
20. 同日上市        — 内容含"股票将于...上市交易"
21. 区域新闻关联    — 标题含省/市地名
22. 行业综述        — 标题含"概览/动态/数据榜"
23. IPO申报关联     — 内容含"首发申请/拟IPO"
24. 同行对比        — 内容含"同行/可比公司"
25. 品牌商标关联    — 内容含"假冒/商标/品牌"
26. 破产债权关联    — 内容含"破产/债权人/清偿"
27. 股权转让        — 内容含"转让占/股权转让"
28. 检验检测关联    — 标题含"检验检测"
29. 同行产品列举    — 内容含"发动机/配套/供应商"
30. 列举式并列提及  — 证据句含3个以上顿号（默认兜底1）
31. 同事件提及      — 无法归入以上任何类型（最终兜底）
```

## 七、已知局限

1. **规则而非模型**：当前分类基于关键词规则，对非标准表述可能漏判或误判
2. **剩余6条未细分**：2.8%的关系仍为"同事件提及"，需要人工判断或LLM辅助
3. **列举式并列提及偏多**（35条）：这些公司只是同在一个名单里，严格来说关联度较弱，可考虑标记为"弱关联"
4. **双向镜像未去重**：A↔B和B↔A各一条，实际是同一关系，展示时可去重
